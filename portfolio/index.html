The project features live search, starting-letter filtering, and alphabetical sorting, all implemented using vanilla JavaScript with a modern, responsive UI built in Tailwind CSS. This project showcases my ability to work with DOM manipulation, event handling, and array methods while delivering a smooth user experience without page reloads. It reflects my focus on clean code, responsive design, and building intuitive user interfaces.
